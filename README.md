# grifon_aegis
Финансовый щит заёмщика — анализ кредитного риска на основе реальных данных 255 000 заёмщиков

Aegis — Финансовый щит заёмщика
Все кредитные инструменты созданы для банков. Aegis создан для людей.
Платформа помогает обычному человеку ответить на главный вопрос перед оформлением кредита: "Я реально потяну этот кредит?" — до того, как банк примет решение за него.
Пользователь вводит свои данные: доход, расходы, желаемую сумму и срок кредита. Система анализирует реальные данные 255 000 заёмщиков, находит людей с похожим финансовым профилем и показывает: какой процент из них не смог выплатить кредит, какова реальная долговая нагрузка, и что произойдёт если доход упадёт или ставка вырастет.
Ключевые функции:

Персональный риск-скор от 0 до 100 с вердиктом 🟢🟡🔴
Поиск похожих заёмщиков по реальному датасету
Стресс-тест: потеря дохода -30%, рост ставки +3%, экстренные расходы
Персональные рекомендации: безопасная сумма кредита для вашего профиля
Аналитический дашборд на основе реальных данных

Технологии: Python, Flask, SQLite, Chart.js. Датасет: 255 347 реальных анонимизированных кредитных записей (Loan Default Dataset, Kaggle).
